### 5th Course in R Programming Specialization by Microsoft ###
## Learning Objectives of the Course: ##

- Collect online data through ethical web scraping and API requests

- Manipulate and summarize large datasets using data.table

- Build interactive geographic visualizations with leaflet

- Create dynamic, user-driven reports using RMarkdown

- Work with Azure Blob storage to read and write data in the cloud

- Use GitHub to share and present your work

## Abilities Gained: ##

- Automate complex data tasks

- Create interactive tools and reports for stakeholders

- Manage data in cloud environments

- Demonstrate your work in a GitHub-based portfolio
